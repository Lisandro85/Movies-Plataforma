Durante mi proceso de aprendizaje, desarrollé este proyecto web utilizando HTML, CSS, Bootstrap y JavaScript para mi frontend; y Express, Axios, MongoDB mancomunadamente con JS para mi backend. Comencé creando la estructura básica con HTML, lo que me ayudó a entender cómo organizar el contenido de una página. Luego, utilicé CSS y Bootstrap para diseñar y hacer que el sitio fuera visualmente atractivo y responsivo. Finalmente, implementé JavaScript, Express, Axios, MongoDB para trabajar con los datos del servidor. Este proyecto me permitió consolidar mis conocimientos y ver cómo las diferentes tecnologías trabajan juntas para crear un sitio web completo y funcional. Fue una experiencia desafiante pero muy gratificante.
